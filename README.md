# Dealora Site

Affiliate product website for trending tech gadgets and budget deals.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Important

Replace these placeholder links in `src/App.jsx` with your real Amazon affiliate links:

- https://amzn.to/your-jbl-link
- https://amzn.to/your-led-link
- https://amzn.to/your-stand-link
- https://amzn.to/your-speaker-link

- https://amzn.to/your-cable-link
